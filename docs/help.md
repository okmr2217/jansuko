## 基本的な使い方

### セクションとは

セクションは、1回の麻雀セッション（複数ゲームのまとまり）を管理する単位です。例えば「2024年1月1日の新年会」のように、その日の対局をまとめて記録できます。

### 点数記録の流れ

1. **セクションを作成する**
   - 参加者（3〜4人）を選択
   - 開始点（例: 35,000点）を設定
   - レート（例: 50円/1,000点）を設定

2. **ゲームごとに点数を入力する**
   - 半荘が終わるたびに各プレイヤーの点数を入力
   - 点数の合計が自動でチェックされます

3. **結果を確認する**
   - 合計点、増減、精算額、順位が自動計算されます

4. **セクションを終了する**
   - 全ゲームが終わったらセクションを終了（締め処理）

---

## 計算ロジック

### 計算式

| 項目 | 計算式 |
|------|--------|
| 合計点 | 各ゲームの点数の総和 |
| 増減 | 合計点 − (開始点 × ゲーム数) |
| 精算額 | 増減 × レート ÷ 1,000 |
| 順位 | セクション内の合計点による順位（1位が最高） |

### 端数処理

- **点数**: 100点単位で記録
- **精算額**: 小数点以下切り捨て

### 計算例

開始点35,000点、3人麻雀、レート50円/1,000点、5ゲーム実施の場合：

| | かいと | きむら | えいご |
|---|---:|---:|---:|
| 1ゲーム目 | -15,000 | 61,000 | 59,000 |
| 2ゲーム目 | 57,000 | 50,000 | -2,000 |
| 3ゲーム目 | 75,000 | 36,000 | -6,000 |
| 4ゲーム目 | 62,000 | 56,000 | -13,000 |
| 5ゲーム目 | 70,000 | 27,000 | 8,000 |
| **合計** | 249,000 | 230,000 | 46,000 |
| **増減** | +74,000 | +55,000 | -129,000 |
| **精算** | +¥3,700 | +¥2,750 | -¥6,450 |
| **順位** | 1位 | 2位 | 3位 |

## 各ページの説明

### セクション一覧

- 進行中・終了済みのセクションを一覧表示
- フィルター、検索、並び替えが可能
- 「新規作成」ボタンからセクションを作成

### セクション詳細

- ゲームごとの点数一覧を表示
- 点数の入力・修正・削除が可能
- 合計点・増減・精算額・順位を自動計算

### 雀士一覧

- 登録されている雀士の一覧を表示
- 管理者のみ雀士の作成・編集・削除が可能

### 統計

- 期間を指定して統計情報を表示
- 勝率、平均順位、通算収支、参加ゲーム数などを確認

### アカウント設定

- 自分の表示名・パスワードを変更

## 権限管理

### ユーザーの種類

| 種類 | 説明 |
|------|------|
| 管理者 | 雀士の作成・編集・削除ができるユーザー |
| 一般ユーザー | 通常の操作が可能なユーザー |
| セクション作成者 | セクションを作成したユーザー |
| セクション参加者 | セクションに参加しているユーザー |

### 権限一覧

#### セクション管理

| 操作 | 一般ユーザー | 管理者 | 備考 |
|------|:------------:|:------:|------|
| セクション作成 | ✅ | ✅ | ログインユーザー全員 |
| セクション編集 | ✅ | ✅ | 作成者のみ |
| セクション終了（締め処理） | ✅ | ✅ | 作成者のみ |
| セクション削除 | ✅ | ✅ | 作成者のみ |

#### 点数管理

| 操作 | 一般ユーザー | 管理者 | 備考 |
|------|:------------:|:------:|------|
| 点数入力 | ✅ | ✅ | 参加者のみ |
| 点数修正 | ✅ | ✅ | 参加者のみ |
| 点数削除 | ✅ | ✅ | 作成者のみ |

#### 雀士管理

| 操作 | 一般ユーザー | 管理者 | 備考 |
|------|:------------:|:------:|------|
| 雀士一覧の閲覧 | ✅ | ✅ | 全員 |
| 雀士の作成 | ❌ | ✅ | 管理者のみ |
| 雀士の編集（表示名） | ❌ | ✅ | 管理者のみ |
| 雀士の削除 | ❌ | ✅ | 管理者のみ |

#### アカウント設定

| 操作 | 一般ユーザー | 管理者 | 備考 |
|------|:------------:|:------:|------|
| 自分の表示名変更 | ✅ | ✅ | 全員 |
| 自分のパスワード変更 | ✅ | ✅ | 全員 |

### 補足

- 管理者であっても、自分が作成者・参加者でないセクションの点数操作はできません
- セクション作成者は自動的にそのセクションの参加者となります

## よくある質問

### Q. 点数を間違えて入力してしまいました

セクション詳細ページから該当のゲームを選択し、点数を修正できます。ただし、修正できるのはセクション参加者のみです。

### Q. セクションを削除したい

セクション詳細ページの設定から削除できます。ただし、削除できるのはセクション作成者のみです。削除すると、そのセクションに関連するすべてのゲーム・点数データも削除されます。

### Q. 新しい雀士を追加したい

管理者に依頼してください。雀士一覧ページから管理者のみ追加できます。

### Q. パスワードを忘れてしまいました

管理者に連絡してパスワードをリセットしてもらってください。

### Q. 3人麻雀と4人麻雀の両方に対応していますか？

はい、セクション作成時に参加人数（3人または4人）を選択できます。

### Q. 点数の合計が合わないとエラーになります

入力した点数の合計が「参加人数 × 開始点」と一致する必要があります。例えば、3人麻雀で開始点35,000点の場合、合計が105,000点になる必要があります。
