│  [ロゴ]  セクション  雀士  統計      [ユーザー名 ▼]    │

## デザインガイドライン

### カラーパレット

|用途|色|
|---|---|
|プライマリ|緑系（麻雀牌をイメージ）|
|成功/プラス|緑|
|エラー/マイナス|赤|
|進行中|緑のバッジ|
|終了済み|グレーのバッジ|
|背景|白またはライトグレー|

### コンポーネント

- ボタン: 角丸、ホバーエフェクト
- カード: 影付き、角丸
- テーブル: ストライプ背景、ホバーハイライト
- フォーム: フローティングラベルまたは上部ラベル
- モーダル: 背景暗転、中央配置

### レスポンシブ対応

- モバイル: シングルカラム、ハンバーガーメニュー
- タブレット: 2カラム可
- デスクトップ: フル表示
