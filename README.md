# じゃんスコ

麻雀の点数記録・精算を簡単に管理できる Web アプリです。

従来スプレッドシートで行っていた面倒な計算を自動化し、ゲームごとの点数入力だけで合計点・増減・精算額・順位をリアルタイムに表示します。

## スクリーンショット

> スクリーンショットのデータは実際に行われたものではなく架空のものです。

### セクション一覧

![セクション一覧](/public/readme/セクション一覧.png)

過去のセクション（ゲームセッション）を一覧で確認できます。各セクションの参加者、日付、レートが表示され、タップして詳細画面へ移動できます。

### セクション詳細

![セクション詳細](/public/readme/セクション詳細.png)

セクション内の各ゲームの点数を記録・管理する画面です。ゲームごとの点数入力に加え、合計点・増減・精算額・順位がリアルタイムで自動計算されます。

### 統計

![統計](/public/readme/統計.png)

プレイヤーごとの成績を確認できます。勝率、平均順位、通算収支、参加ゲーム数などを期間別（全期間/月別/カスタム）に表示します。

## 主な機能

- **セクション管理** - 参加者・開始点・レートを設定してゲームセッションを作成
- **点数記録** - ゲーム終了ごとに各プレイヤーの点数を入力（バリデーション付き）
- **自動計算** - 合計点、増減、精算額、順位を自動算出
- **統計機能** - 勝率、平均順位、通算収支などを期間別に確認
- **雀士管理** - プレイヤーの登録・管理

## 対応

- 3 人麻雀 / 4 人麻雀
- モバイル対応（スマホからでも使いやすい）

## 技術スタック

- **フレームワーク**: Next.js
- **スタイル**: Tailwind CSS
- **UI ライブラリ**: shadcn/ui
- **DB・認証**: Supabase

## Getting Started

開発サーバーを起動:

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。
